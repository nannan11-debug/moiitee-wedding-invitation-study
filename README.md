# 모이티 (Moiitee) Wedding Invitation - Study Clone

This is a local study/research clone of a Korean mobile wedding invitation from [Moiitee](https://www.moiitee.com/invite/myeon).

## Purpose
Educational and research purposes only. **Not for commercial use.**

## What's Included
- Rendered HTML page with all content
- CSS stylesheets (10 files)
- JavaScript bundles (6 files)
- Korean fonts (Pretendard, NanumMyeongjo, etc.)
- Wedding photos and decorations
- SVG icons and UI assets

## Notes
- Some external resources (Kakao SDK, Google Fonts) still reference online CDNs
- The page is a React SPA - content is rendered client-side
- Original site: https://www.moiitee.com/invite/myeon

## Tech Stack
- React (SPA)
- Pretendard font family
- Kakao Maps/Share SDK
- CloudFront CDN for media assets
